JAMAL.ICT Portfolio Asset Generator Dashboard

How to use this Generator:

1. Open this file (`generator.html`) in any modern web browser (Chrome, Edge, Firefox).

2. Click each button below to dynamically build and download the exact placeholder files required by your portfolio.

3. Move the downloaded items into your local project folders (`assets/images/` and `assets/documents/`).

Images Asset Bundle (`assets/images/`)

Generates dark-tech visual templates styled specifically to your theme profile palette.

Document PDF Bundle (`assets/documents/`)

Generates clean structural sample documents mapped directly to your routing layout links.
